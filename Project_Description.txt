# Downloads Sorter v1.1.2
A lightweight, open-source PowerShell tool to automatically organize your 
cluttered Windows Downloads folder.

## Key Features
- **Smart Sorting:** Automatically categorizes files into subfolders.
- **Improved Auto-Cleanup:** Choice to run the cleanup automatically every 
  6 hours (starts at 6:00 AM) with a more compatible task trigger.
- **Resilient Scheduling:** Task is set to run as soon as possible if a 
  scheduled run is missed.
- **Detailed Logging:** Track all moved files in 'SortDownloads.log'.
- **Custom Icon:** Includes the "Coffee & Folder" icon.
- **Easy Installer:** Custom-built C# installer for one-click setup.

## Installation
1. Download the latest release ZIP.
2. Extract all files to a folder.
3. Run Installer.exe.
4. (Optional) Select 'Yes' when prompted for automatic 6-hour cleanup.
